Etik och Integritet inom Webben

När får staten ta del av vår elektroniska kommunikation och var går gränsen för staglig kontroll?

Att använda sig av webben är något väldigt speciellt eftersom man kan använda den 24 timmar om dygnet samtidigt som man kan söka upp i princip vad som helst och få ett svar. Men hur hade det känts ifall allt du sökte upp och kollade på hade kunnat bli sedd av staten. Samtidigt hade kanske friheten att kunna söka upp vad som helst tagits bort från en. Tänk om alla ens telefonsamtal hade blivit inspelade så att staten skulle kunna lyssna på vad man pratade om. Vad går egentligen gränsen för statlig kontroll och när får polisen ta del av vår elektroniska kommunikation?

Integritet är ett viktigt ämne för många då dem vill kunna känna att dem har en frihet om att kunna göra vad dem vill utan att staten eller någon ska kontrollera detta. Dem flesta vill alltså inte att någon främling ska kunna veta vad man gör 24 timmar om dygnet. Dock kan man inte vara för fri eftersom, utan kontroll kan det ske total anarki. Staten måste alltså kunna se till att varje individ känner att dem har tillräckligt mycket integritet och frihet, samtidigt som dem måste kontrollera brottslingarna.

Det är fall som, ifall staten misstänker att någon kan vara en terrorist som planerar ett terrordåd. Då måste staten samt polisen göra allt dem kan för att se till att detta terrordådet inte sker. Detta kan leda till att dem behöver bevaka den här personen alla timmar om dygnet samtidigt som dem måste lyssna på dess samtal för att få reda på ens planer. Saker som dessa är dock bara tillåtet ifall domstolen tillåter det eller är en sådan farlig situation så att det krävs (enligt PTS). Alltså kan detta bara ske ifall staten anser att man är farlig som ser till att inte vem som helst kan bli övervakad eller avlyssnad. Samtidigt finns det också lagar, som GDPR och LEK, som skyddar ens integritet och personliga uppgifter så att dem inte kan spridas till vem som helst.

Etik och Integritet inom webben

Nuförtiden är det lättare än någonsin att publicera information som fotografi på internet som olika sociala medierna. Samtidigt gör detta också så att denna information kan spridas väldigt snabbt så att fler kan se på ens inlägg och ge den en “like”. Dock finns det fortfarande etiska problem med detta då någon kan publicera bilder av dig som du inte vill ska vara ute på internet. När man tagit bort inlägget så kan det redan ha spridits sig så att massvis med människor har sett det samt sparat inlägget på deras egna elektroniska apparater. Då kommer det vara nästan omöjligt att få bort från internet igen.

Vid situationer som dessa så kan det leda till konsekvenser som fängelsestraff ifall den utsatta var utan kläder (naken). Det kan också vara mindre saker som att man blir ovän med någon som kan avsluta vänskap. Därför är det väldigt viktigt att se till att man frågar innan man publicerar fotona. Självklart finns det fortfarande situationer då det är mindre viktigt att fråga personerna i bild ifall man får publicera den. Detta är i situationer som på gatu torg där det är naturligt att ta bilder samtidigt som man oftast inte försöker fotografera en enskild person (dock bör man fråga ifall man gör så).

Detta är varför lagar som LEK och GDPR finns som är där för att skydda din privata information samt integritet. Med hjälp av dessa, ser man till att information som man inte vill ska vara ute på internet, inte är ute på internet heller.
