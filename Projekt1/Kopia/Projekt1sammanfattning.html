Sammanfattning

Vad är webben?

Webben är en plats där information kan samlas, laddas upp, skapas, och besökas på en dator. På webben finns det massvis med webbsidor där informationen kan finnas. Alltså är webbsidorna kopplade och skapade i webben. World Wide Web är namnet på den populäraste webben och för kortast www som också skrivs i början av alla webbsidor där (www.).

Skillnaden mellan Internet och Webben

Som sagt tidigare är webben en plats där information kan samlas och besökas. Internet Internet är istället hur datorer kontaktar och kopplar sig till varandra. Innan internet hade skapats så behövde man en direkt koppling till båda datorerna för att kunna dela information mellan dem. Man kan säga att det fungerade som ett telefonsamtal på den tiden. Skillnaden mellan internet och webben är att internet är grunden som webben befinner sig på. Internet är ett gigantiskt nätverk som håller samman webben och dess användare medans webben innehåller informationen i form av webbsidor som kan nås genom internet. Alltså fungerar inte webben utan att internet finns.

Olika begrepp

HTML står för Hyper Text Markup Language och skapades 1989 av Tim Berners Lee som använde detta språk för att skapa webben.

CSS står för Cascading Style Sheets och kopplas till HTML webbsidor och används för att designa dessa webbsidor.

HTTP står för Hyper Text Transfer Protocol och är ett protokoll som tillåter utbytet av HTML filer och hemsidor mellan servers och webbläsare. HTTP används än idag fast med en mer uppdaterad version än den man använde under 90 talet. Det har också kommit ut säkrare versioner av HTTP som kallas HTTPS och står för Hyper Text Transfer Protocol Secure.

IP står för Internet Protocol och är ett sorts nummer som används för att identifiera olika enheter på internet. Man brukar också kalla detta en IP adress och är väldigt viktig för att olika enheter ska kunna kopplas till internet samt kopplas tillsammans.

TCP står för Transmission Control Protocol är ett nätverk Protokoll som tillåter två enheter att ansluta till varandra och utbyta data med varandra. TCP är ett bra exempel av en funktion som behöver en IP adress för att kunna fungera och ansluta två enheter till varandra.

När man skickar ett mail med en fil används ofta system som FTP som står för File Transfer Protocol som används för att ladda upp och överföra filer. För att detta ska fungera behövs både SMTP (Simple Mail Transfer Protocol) och DNS (Domain Name System) servern för att hitta saker som IP adresser.

Vilkad skapade internet och hur spreds det?

1969

Advanced Research Project Agency Network eller ARPANET var ett datornät i usa tillkommet 1969. Det skapades för att främja forskning kring just datornät och det var världens första fungerande paket växlande nätverk. Det skapades av försvarets utvecklingsavdelning i USA under kalla kriget som mycket annan teknik och var början för internet. Världens första meddelande över internet skickades via ARPANET med meddelandet “login” men innehöll bara bokstäverna L och O eftersom mottagar datorn kraschade efter att ha tagit emot O:et.

1971-1974

1971 skickades det första e-mailet av Ray Tomlinson. Några år senare, 1974, så skapades TCP eller Transmission Control Protocol och IP adresser. Dessa skapades så att man kan koppla olika datorer till varann och dela information.

1985

Toppdomän eller TLD kom 1985 och delas upp i 3 kategorier, Nationella toppdomäner (ccTLD), Generiska toppdomäner (gTLD) och infrastrukturella toppdomäner. Toppdomän, förkortad TLD, är den högsta nivån i internets domännamnssystem (DNS). Toppdomäner infördes 1985 för att decentralisera tilldelandet av domänadresser, så att till exempel varje land kunde själva tilldela domänadresser till sina organisationer och medborgare. Innan dess fanns det endast en enda lista domäner.

1991

HTML som står för HyperText Markup Language var ett märkspråk som skapades/publicerades 1991 och används för att skapa webbsidor. HTML kan också paras upp med CSS som står för Cascading Style Sheets fast skapades/publicerades inte förens 1996.

1992-Idag

Dial-up internet erbjöds kommersiellt först 1992. Då användes ett modem som kopplades till telefonnätet. För att kunna använda internet genom dial-up behövde man ringa internetleverantörer som gav en tillgång till internet under en viss tid. Bredband kom till vid 2000-talet och var den nyare versionen av internet delning. Istället för att behöva ringa leverantörerna för att få tillgång till internet så var bredbandet alltid kopplad till internet så att alla skulle få uppkoppling utan att ringa någonstans. Idag använder man oftast fiber för att få koppling till internet eftersom kopplingen sker mycket snabbare än via bredband. Under tiden fiber kom till så började också mobilt bredband användas så att man skulle kunna ha internetuppkoppling från varsomhelst.

Hur används webben idag jämfört med förr i tiden?

1989 var webbens början då Tim Berners Lee och Robert Cailliau gav ett förslag till CERN där dem jobbade. Anledningen till varför dem ville skapa internet var så att man skulle kunna dela och bläddra igenom massvis med information genom en dator. Detta gjordes via hypertext som heter Hyper text markup language eller HTML som är ett språk som används bortom vanlig text. 1990 lanserades organisationen World Wide Web (WWW) men internet blev fortfarande inte populärt innan sökmotorer som Mosaic och Netscape skapades.

Webbdesign 1.0 var den första stora versionen av webbdesign och stod ut med sin tråkiga gråa bakgrund, simpel text och de få variationerna mellan webbsidor. På denna tiden hade man inte lika stort intresse av att tjäna pengar från sina webbsidor så man såg sällan reklam på webben. Detta ändrades med webbdesign 2.0 där man började tänka mer på användaren när man designade webbsidorna. På denna tid var webbsidorna mycket bättre designade och smartare eftersom företag fick fram idéen att man kunde tjäna pengar från webbsidorna man skapade. När webbdesign 3.0 publicerades så blev webben mycket smartare när man började använda artificiell intelligens tillsammans med webben. Detta gjorde det mycket lättare och användarvänligt att använda sökmotorer med webbdesign 3.0.

Sociala Medier genom tiderna

Den första sociala media sajten som för internet var “Six Degrees” vilket var en sajt gjord vid webbdesign 1.0. Där kunde man skapa en profil som kunde visas till dina vänner. Detta var den första social media sajten som senare har inspirerat de sociala medierna vi använder oss av idag.

Under 2000-talet började sociala medier kraftigt öka i popularitet. Många av dessa hade olika specifika användningsområden som Photobucket och flicker som fokuserade på fotodelning medans Youtube lät folk dela videor på ett sätt man inte kunde tidigare. År 2006 var året då några av dagens största sociala nätverk skapades som Facebook och Twitter. med Web 2.0 så ökade populariteten av internet som samtidigt ökade antalet användare av sociala medier. Detta ledde då till att flera stora sociala medier skapades med tanken av att försöka tjäna pengar från det som till exempel, Instagram. Än idag är Instagram, Twitter, Facebook och Youtube fortfarande väldigt populära med miljontals användare och förbättrar fortfarande deras versioner för att kunna tjäna ännu mer pengar.

Webbläsare, HTML och sökmotorer genom tiderna

Webbläsare

Webbläsare är ett program för att hämta och tolka kodade dokument. Dessa koder används för att skapa webbsidor och läggs upp på olika webbservrar på internet. Den första webbläsaren utvecklades av Tim Berners Lee och lanserades 1993 som kallades för Mosaic vid NCSA i Illinois. Detta var revolutionerande eftersom det var den första grafiska webbläsare som hade bilder och ljud som allmänheten kunde få tag på. Web användandet ökade väldigt mycket som ledde till att fler och fler webbläsare skapades senare under 90 talet där dem krigade om popularitet. En av dem mest revolutionerande webbläsare som också är världen mest populära webbläsare nuförtiden är Google Chrome. Vad som är annorlunda med denna är att den hade en massa funktioner och google appar inbyggda i webbläsaren. Detta gjorde att operativsystemen, som windows, får mindre betydelse eftersom man kan komma åt de vanligaste användningsområden genom webbläsaren nu.

Vad är HTML?

HTML står för HyperText Markup Language och är ett märkspråk som används för att skapa och organisera webbsidor och skapades av Tim Berners Lee i 1989. HTML är inte ett programmeringsspråk och har inte möjligheten att skapa dynamiska funktionsförmåga, istället fungerar HTML på ett sätt som liknar Microsoft Word.

De Olika versionerna av HTML

Den första versionen av HTML publicerades 1991 och hade totalt bara 18 tags, men nu har den nyaste versionen över 100 tags totalt som visar hur mycket HTML har utvecklats. Totalt finns det 7 större HTML versioner som publicerades 1991, 1994, 1995, 1997, 1999 och 2014. Alla dessa versioner har förbättrat koden inom HTML och både skapat och tagit bort flera tags. Många tags som inte behövdes längre inom HTML flyttades över till CSS som står för Cascading Style Sheets och används för att design HTML webbsidor. Dem nyaste versionerna av HTML och CSS heter HTML 5 som publicerades 2014 och CSS level 3 som publicerades 1999.
